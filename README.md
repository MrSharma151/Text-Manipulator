
# Text Manipulator

Welcome to Text Manipulator, a React application designed to simplify text transformation tasks with the added convenience of a dark mode feature.

## Features:

- **Text Transformation:** Easily transform text with various options such as uppercasing, lowercasing, capitalizing, and more.
- **Dark Mode Support:** Enhance your viewing experience with a sleek dark mode interface.
- **Responsive Design:** Enjoy seamless usability across different devices, ensuring a consistent experience.

## Technologies Used:

- **React:** Utilize the power of React for building dynamic user interfaces.
- **CSS:** Style the application and implement the dark mode feature with CSS.
- **JavaScript:** Leverage JavaScript for text manipulation functionalities.

## Getting Started:

To get started with Text Manipulator, follow these simple steps:

1. **Clone the Repository:** `git clone https://github.com/your-username/text-manipulator.git`
2. **Navigate to the Project Directory:** `cd text-manipulator`
3. **Install Dependencies:** `npm install` or `yarn install`
4. **Run the Development Server:** `npm start` or `yarn start`
5. **Open the Application:** Visit `http://localhost:3000` in your web browser.

## Usage:

1. Enter or paste your text into the input field.
2. Choose the desired transformation option from the provided buttons.
3. Enjoy the transformed text instantly!
4. Toggle between light and dark mode using the provided switch.
